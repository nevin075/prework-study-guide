HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element has 2 attributes src(its location) & alt(alt text)
  • Most elements have an opening and closing tag ex. " < p > content < / p > "
  • Elements also have attributes which contain extra info or alter an element
  • HTML can be use to do basic mark up (this can be important for screen readers)
  • HTML defines 6 levels headings, H1-H6, H1 being most important and H6 the least

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stage changes
  • git commit -m "commit message here": commit changes
  • git pull origin main: pulls base branch
  • git push origin name/of-branch: push changes made in branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • A function is a reusable portion of a program
  • Control flow is the order in which a computer executes code in a script.
  • Use "if (conditon) {execute this code}" to specify a block of code to be executed, if a specified condition is true
  • Use "else (conditon) {execute this code}" to specify a block of code to be executed, if the same condition is false
  • Use "else if (conditon) {execute this code}" to specify a new condition to test, if the first condition is false
  • Use a "for (initialization; termination;increment) {execute this code}" loop when you know exactly how many times you want to loop through a block of code
    • ex. for(int i=1; i<11; i++){ System.out.println("Count is: " + i);
